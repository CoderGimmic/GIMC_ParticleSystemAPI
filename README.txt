===============================================================================
  1. Introducion
===============================================================================

  The GIMC Particle System API is divided into two projects.

  The library which is "GIMC_ParticleSystem.sln" and the demos which is 
  "GIMC_ParticleSystem_demos.sln". 

  The libraries are required to run the demos found in the "demos/bin" folder.


===============================================================================
  2. Building the Library
===============================================================================

  Build the "GIMC_ParticleSystem_debug.lib" and "GIMC_ParticleSystem.lib" 
  libraries using the "GIMC_ParticleSystem.sln" solution. 


===============================================================================
  2. Running the Demos
===============================================================================

  Run the demos from the "GIMC_ParticleSystem_demos.sln" solution located in
  the "Projects/vc2012" directory.
